# Installation

## Stable release

To install Python API Client for FDMS API Gateway, run this command in your terminal:

```sh
uv add zfdms
```

Or if you prefer to use `pip`:

```sh
pip install zfdms
```

## From source

The source files for Python API Client for FDMS API Gateway can be downloaded from the [Github repo](https://github.com/ronnzw/zimra_zfdms).

You can either clone the public repository:

```sh
git clone git://github.com/ronnzw/zimra_zfdms
```

Or download the [tarball](https://github.com/ronnzw/zimra_zfdms/tarball/main):

```sh
curl -OJL https://github.com/ronnzw/zimra_zfdms/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd zimra_zfdms
uv pip install .
```
